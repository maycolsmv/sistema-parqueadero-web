# 🚗 Parking Management Web System

Academic project focused on web development.

## 🔐 Features
- User authentication with secure password hashing
- Vehicle registration and search
- Real-time parking space status (occupied / available)
- Relational database integration

## 🛠️ Technologies
- PHP
- MySQL
- JavaScript
- HTML / CSS

## 🎯 Project Goal
To develop a web application that efficiently manages parking space usage and vehicle records.
